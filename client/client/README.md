# FinGPT Client

React frontend for the FinGPT personal finance tracker. This client uses Vite.

## Scripts

```powershell
npm install
npm start
```

Runs the client at `http://localhost:3000`.

```powershell
npm test
```

Runs the Vitest test suite.

```powershell
npm run build
```

Builds production assets to `dist/`.
